# Workshop Agenda

## AI Evals Workshop - Allegro x GenAI in Localisation

Date: TBD  
Duration: 3.5 hours (including breaks)

---

## 1. Welcome and Context (15 min)

- Introductions and workshop goals
- Why evaluation matters in AI-assisted localisation
- Overview of today’s deliverables

## 2. Framework Briefing: Error Taxonomy and Severity (25 min)

- Accuracy error subtypes used during annotation
- Severity levels and decision boundaries
- Examples: acceptable vs. critical translation issues

## 3. Live Calibration Exercise (35 min)

- Group annotation of sample e-commerce segments
- Discussion of edge cases and disagreement patterns
- Alignment on annotation standards for the workshop

## 4. Break (10 min)

## 5. Dataset Annotation Sprint (40 min)

- Individual/small-group annotation of assigned dataset slices
- Quality checks and consistency tips
- Export and handoff format review

## 6. Prompt Experimentation (40 min)

- Baseline prompt setup
- Iterative prompt refinement based on annotation goals
- Capturing prompt versions and rationale

## 7. Evaluation Run in LangSmith (30 min)

- Configure evaluation criteria and run settings
- Execute structured evaluation
- Review metrics and representative failures

## 8. Results Review and Workflow Improvements (20 min)

- Identify failure modes by category/severity
- Translate findings into prompt and process improvements
- Define next iteration priorities

## 9. Q&A and Wrap-Up (15 min)

- Open questions
- Recommended next steps for teams
- Reference resources and follow-up materials

---

## Materials Checklist

- Access to the invited LangSmith workspace
- Annotated dataset file(s)
- Workshop repository cloned locally
- Spreadsheet editor (Google Sheets or Excel)

## Session Outcomes

By the end of the workshop, participants will have:

- A labelled localisation dataset
- At least one iterated prompt with documented changes
- An evaluation run with interpretable outputs
- A short action plan for improving translation quality workflows
